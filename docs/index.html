Organizador efectivo para uso diario de manera personal. ¡DESCARGALO YA!

Perfecta y Cómoda Interfaz

Modelo de diseño intuitivo, para las anotaciones de viajes, trabajo, horario etc.

Fácil Colaboración

Puedes compartir tus eventos personales, empresariales o educativos con las personas mas cercanas y que todos conozcan tus aventuras

Maneja tus proyectos

Un proyecto es un objetivo combinado con un conjunto de tareas que debes completar para conseguirlo. Crea tantos proyectos como quieras para anotar todos los resultados deseado.

Organiza tus metas

Cree un número ilimitado de proyectos y agrúpelos por categorías mediante carpetas.

Organiza tus tareas

Organiza las tareas de diferentes proyectos utilizando listas de contexto flexibles. Si estás familiarizado con GTD te encantará esta función.

Planea tu día

Establece fechas de vencimiento para las tareas y haz planes para cualquier día en particular.

Sincroniza tus datos

Chaos Control funciona tanto en el escritorio como en los dispositivos móviles. Crea una cuenta y sincroniza tus proyectos en todos tus dispositivos

Comparte tus eventos

Coloca todas las tareas, notas e ideas entrantes en Chaos Box para procesarlas más tarde. Funciona de forma similar a la bandeja de entrada de GTD, pero puedes utilizarla como una simple lista de tareas.
